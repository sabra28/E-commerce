# E-commerce projet
========================
E-commerce web application.

Requirements
------------

  * PHP 7.1.3 or higher;
  * Symfony 5 
  * Wamp (PHPMYADMIN)

Installation
------------

```bash
git clone https://github.com/Antoine87/symfony-ecommerce.git
cd symfony-ecommerce
composer install
yarn
yarn run dev
```


Usage
-----

```bash
php bin/console server:run
```
