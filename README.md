# E-commerce projet
========================
E-commerce web application.

Requirements
------------

  * PHP 7.1.3 or higher;
  * Symfony 5 
  * Wamp (PHPMYADMIN)

Installation
------------

```bash
git clone https://github.com/sabra28/E-commerce.git
cd symfony-ecommerce
Modifier le .env root:@127.
composer install
php bin/console doctrine:schema:create
```


Usage
-----

```bash
php bin/console server:start
```
